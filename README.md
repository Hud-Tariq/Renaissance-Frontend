## 📖 About The Project

For our **Programming for AI Semester Project**, our team developed an image model capable of deconstructing and predicting the artistic process. The model visualizes the creation of a face sketch through **7 distinct steps**, capturing the evolution from foundational structure to the final polished artwork.

To achieve this, we:
- 🖌️ Curated and created our own custom dataset.
- 🧠 Fine-tuned a high-quality model using advanced deep learning techniques.
- 💻 Built a frontend (this repository) to interact with and showcase the generation steps.
