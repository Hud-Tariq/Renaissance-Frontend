Introducing, Renaissance - The Step by Step Sketch Predictor.
For our Programming for AI Semester Project, me and my groupmates (Uns Ahmad, Musa Mehar, Aqib Hussain) made an image model which predicts the sketch of a face in 7 distinct steps, starting from initial structure to final end result.
We fine tuned a model on our own self made dataset .
Dataset link: "https://www.kaggle.com/datasets/aqibhussainmalik/step-by-step-sketch-predictor-dataset/"
Demo notebook link: "https://www.kaggle.com/code/musamehar/sketch-model"
Trained model link: "https://huggingface.co/HudTariq/loomis-model-output-v4-high-quality-7steps"
